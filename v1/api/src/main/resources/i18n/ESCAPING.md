# Encoding

```
Code should be prepended with backslash \.
Example: 200 -> \200 
```

| Letter | ISO 8859-2 | Unicode |
|--------|------------|---------|
| č      | 232        | u010D   |
| š      | 185        | u0161   |
| ž      | 190        | u017E   |
| Č      | 200        | u010C   |
| Š      | 169        | u0160   |
| Ž      | 174        | u017D   |
